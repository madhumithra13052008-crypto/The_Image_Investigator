# 🕵️ THE IMAGE INVESTIGATOR

**The Image Investigator** is a desktop-based image editing and annotation application developed using **Python**, **OpenCV**, **Tkinter**, and **Pillow**. It provides an intuitive graphical interface that allows users to collect, preview, edit, annotate, and save images efficiently.

This project was developed to explore image processing concepts while building a user-friendly desktop application with practical image editing features.

---

## ✨ Features

- 📂 Open images from your local system
- 🖼️ Live image preview
- 🔄 Rotate images
- ↔️ Flip images (Horizontal & Vertical)
- ✂️ Crop images
- ⭕ Draw circles
- ▭ Draw rectangles
- ➡️ Draw arrows
- ✏️ Annotate images
- ↩️ Undo edits
- ↪️ Redo edits
- 💾 Save edited images
- 🖥️ Interactive graphical user interface

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/THE-IMAGE-INVESTIGATOR.git
```

### 2. Navigate to the project folder

```bash
cd THE-IMAGE-INVESTIGATOR
```

### 3. Install the required dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python main.py
```

## 📖 How It Works

1. Launch the application.
2. Open an image from your computer.
3. Preview the selected image.
4. Choose an editing or annotation tool.
5. Apply the desired modifications.
6. Use Undo or Redo if needed.
7. Save the edited image.
